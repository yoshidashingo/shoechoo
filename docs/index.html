Shoe Choo

A distraction-free Markdown editor for macOS. Live preview, focus mode, and a clean writing experience.

Download GitHub

Seamless Editing

  • Live preview — Markdown is rendered inline as you type, just like Typora
  • Clean, minimal interface that stays out of your way
  • Subtle syntax highlighting without visual clutter

Writing Focus

  • Focus mode dims everything except the current paragraph
  • Typewriter scrolling keeps the active line centered on screen
  • Immersive full-screen writing with no distractions

Markdown

  • Full Markdown support — headings, lists, tables, code blocks, math (LaTeX), footnotes
  • Drag & drop or paste images directly into your document
  • Export as Markdown, HTML, or PDF

General

  • Native macOS app built with Swift & SwiftUI
  • Full support for light and dark mode
  • Low memory usage, instant launch

Installation

Download from GitHub Releases, unzip, move to Applications, and launch.

Requires macOS 14 (Sonoma) or later.

This app is not notarized. macOS will block it on first launch.

Run in Terminal: xattr -cr /Applications/shoechoo.app

Or: try to open the app → System Settings → Privacy & Security → Open Anyway
