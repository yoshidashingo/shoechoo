集中 (Shoe Choo)

書くことに集中できる、macOS向けのシンプルなマークダウンエディタ。ライブプレビュー、フォーカスモード、クリーンな執筆体験。

ダウンロード GitHub

English · 日本語

シームレスな編集体験

  • 入力と同時にMarkdownがその場でレンダリングされる、Typoraライクな見たまま編集
  • 余計なUIを排除した、文章に集中できるミニマルなデザイン
  • 視覚的にわかりやすく、邪魔にならない繊細なシンタックスハイライト

書くことへの集中

  • フォーカスモードで現在の段落以外を薄く表示、今書いている部分に意識を集中
  • タイプライタースクロールでアクティブな行を画面中央に保持
  • 没入感のあるフルスクリーン執筆環境

Markdown

  • 見出し、リスト、テーブル、コードブロック、数式（LaTeX）、脚注をフルサポート
  • ドラッグ＆ドロップやペーストで画像を直接ドキュメントに挿入
  • Markdown、HTML、PDFとしてエクスポート

全般

  • Swift & SwiftUIで構築されたネイティブmacOSアプリ
  • ライト・ダーク両方の外観に完全対応
  • 低メモリ使用量で瞬時に起動

インストール

GitHub Releases からダウンロードし、解凍後アプリケーションフォルダに移動して起動。

macOS 14 (Sonoma) 以降が必要です。

このアプリは公証（Notarization）されていません。初回起動時にmacOSによってブロックされます。

ターミナルで実行: xattr -cr /Applications/shoechoo.app

または: アプリを開こうとする → システム設定 → プライバシーとセキュリティ → このまま開く

FAQ

セキュリティは大丈夫ですか？
はい。ソースコードはGitHubで全て公開しています。ご自身で確認・ビルドが可能です。
サポートはしてもらえますか？
support@shoechoo.com にご連絡いただけますが、不具合報告や機能追加のご要望はGitHubのプルリクエストのみで受け付けています。
iOS版は出ますか？
はい。iOS / iPadOS版とiCloud同期をロードマップに予定しています。
