集中エディタ

書くことに集中できる、macOS向けのシンプルなMarkdownエディタ。ライブプレビュー、フォーカスモード、クリーンな執筆体験。

v0.1 — macOS 14+

English · 日本語

シームレスな編集体験

  • 入力と同時にMarkdownがその場でレンダリングされる、Typoraライクな見たまま編集
  • 余計なUIを排除した、文章に集中できるミニマルなデザイン
  • Highlightrによるコードブロックのシンタックスハイライト

書くことへの集中

  • フォーカスモードで現在の段落以外を薄く表示、今書いている部分に意識を集中
  • タイプライタースクロールでアクティブな行を画面中央に保持
  • 没入感のあるフルスクリーン執筆環境

Markdown

  • GFMサポート — 見出し、太字、斜体、取り消し線、リスト、タスクリスト、テーブル、コードブロック、引用、リンク、画像
  • ドラッグ＆ドロップやペーストで画像を直接ドキュメントに挿入
  • HTMLまたはPDFとしてエクスポート

全般

  • Swift 6 + SwiftUI + TextKit 2 で構築されたネイティブmacOSアプリ
  • 自動保存、macOS バージョン管理、タブ付きウィンドウ
  • ライト・ダーク両方の外観に完全対応（オーバーライド設定あり）
  • サイドバーで最近使ったファイルにすばやくアクセス
  • フォント、サイズ、行間のカスタマイズ

インストール

GitHub Releases からダウンロードし、解凍後アプリケーションフォルダに移動して起動。

macOS 14 (Sonoma) 以降が必要です。

このアプリは公証（Notarization）されていません。初回起動時にmacOSによってブロックされます。

ターミナルで実行: xattr -cr "/Applications/shoechoo.app"

または: アプリを開こうとする → システム設定 → プライバシーとセキュリティ → このまま開く
