# 株式会社セクションナイン 会社紹介

## 会社紹介の構成

セクションナインの会社紹介は以下のストーリーラインで構成する。

### 1. タイトル

- **キャッチコピー**: クラウドモダナイゼーションの専門技術集団
- **サブタイトル**: クラウドを活かしたモダナイゼーション戦略の策定から実現まで圧倒的な推進力で実現する攻性の技術者集団

### 2. 社名の由来

- Section-9 = 「クライアント内の1技術部門として利用可能な組織」
- 社外のパートナーでありながら、社内チームのように深く入り込み、技術力で事業成長を牽引する姿勢を表現

### 3. 代表プロフィール

- **吉田真吾** (Shingo Yoshida)
- AWS Serverless Hero（2019年〜）
- 2011年頃からAWSを活用したシステム開発・コンサルティングに従事
- 初期はキャンペーンサイト・ソーシャルゲームのインフラ構築
- 現在は企業の基幹システムからSaaS構築まで対応
- ChatGPT Community (JP)、LangChain Community (JP)、Serverless Community (JP) 主宰
- 技術書の著作・監修多数（LangChain、ChatGPT、Dify、AWS サーバーレス関連）

### 4. 事業ドメイン

```
クラウドモダナイゼーション
├── サーバーレス＆モダンアプリケーション開発
├── Platform Engineering支援
├── 生成AIアプリケーション開発
├── AWSコスト削減
└── AWSエンジニア人材紹介（HEROZ）
```

### 5. 強みの訴求ポイント

1. **10年のサーバーレス専業実績**: 自社・クライアントともにサーバーレスアーキテクチャで構築
2. **コミュニティ駆動**: 3つの技術コミュニティ運営による最新技術の継続的追跡と人材ネットワーク
3. **一気通貫支援**: 戦略策定→設計→開発→運用→コスト最適化のライフサイクル全体をカバー
4. **攻めのIT**: 守りのIT（コスト削減・効率化）だけでなく、事業成長に直結する「攻めのIT」を実現

### 6. 実績紹介

- エイチ・ツー・オー リテイリング: AWSモダナイゼーション
- トークノート: 顧客基盤のモダン化
- ブレイン・ラボ: B2Bサービスのユーザー体験向上

### 7. サービス紹介への導線

各サービスの詳細は service-catalog.md および services/ 配下の個別ファイルを参照。
