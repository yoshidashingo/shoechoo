# HEROZ - AWSエンジニア人材紹介サービス 実施要領

## サービス概要

AWSに特化したエンジニア人材紹介サービス。
豊富なスタートアップCTO経験者が、企業の技術戦略や組織づくりの戦略を深く理解した上で、コミュニティネットワークを活かしてAWSエキスパートの採用を支援する。

## 対象者

- SRE・クラウドエンジニアを採用したい企業
- LLMアプリケーションエンジニアを確保したい企業
- クラウドネイティブ開発チームを新規に立ち上げたい企業
- 社内にAWS技術リーダーが不在の企業

## 紹介可能な人材カテゴリ

| カテゴリ | スキルセット |
|---------|------------|
| SRE | インフラ自動化、監視設計、インシデント対応、SLI/SLO設計 |
| クラウドエンジニア | AWSアーキテクチャ設計、IaC、サーバーレス開発、コンテナ運用 |
| LLMアプリケーションエンジニア | LangChain/LangGraph、RAG設計、プロンプトエンジニアリング |

## 実施内容

```
1. 人材要件定義
   - 企業の技術戦略・組織戦略のヒアリング
   - 必要な人材像の定義（スキル、経験、役割、カルチャーフィット）
   - 採用要件書の作成

2. 候補者サーチ
   - Section-9コミュニティネットワークからの候補者リストアップ
   - スキルマッチングと適性評価
   - 候補者への打診・動機付け

3. マッチング
   - 企業と候補者の面談設定
   - 双方のフィードバック収集
   - 条件調整サポート

4. オンボーディング支援
   - 入社後の技術キャッチアップ支援
   - Section-9の他サービス（技術支援・研修）との連携
```

## Section-9ネットワークの特徴

- **Serverless Community (JP)** 参加者: サーバーレス技術に精通したエンジニア
- **LangChain Community (JP)** 参加者: LLMアプリケーション開発に実績のあるエンジニア
- **ChatGPT Community (JP)** 参加者: 生成AI活用に関心の高い技術者
- 技術書籍の読者・レビュアーコミュニティ
- クライアントプロジェクトを通じた実務者ネットワーク

## 差別化ポイント

- CTO経験者による技術戦略の深い理解に基づくマッチング
- コミュニティ運営者ならではの、表に出にくい優秀人材へのアクセス
- 採用後のオンボーディングをSection-9の技術支援サービスで補完可能
