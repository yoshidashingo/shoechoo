# Business Overview — shoechoo

## ビジネスコンテキスト

shoechoo は macOS 向けの集中執筆型 Markdown エディタ。Typora に代表される WYSIWYG 方式を採用し、Markdown のソースコードとレンダリング結果を一つのエディタ画面内でシームレスに統合する。

主な価値提案:
- **集中執筆体験**: フォーカスモードとタイプライタースクロールにより、執筆中のブロックに注意を集中
- **WYSIWYG 編集**: 非アクティブブロックではデリミタを非表示、アクティブブロックではソースを表示
- **軽量・ネイティブ**: macOS ネイティブアプリとして動作し、Electron 系エディタより軽量
- **テーマカスタマイズ**: 7 種類のプリセットテーマ

ターゲットユーザー: 技術文書・ブログ・ノートを Markdown で執筆する macOS ユーザー。

| BT-11 | HTML エクスポート | Markdown → HTML 変換しファイル保存 | メニュー / ⇧⌘E |

## ビジネス用語辞書

| 用語 | 定義 | コード上の対応 |
|------|------|---------------|
| ブロック | Markdown ドキュメントの構造単位（段落、見出し、コードブロック等） | `EditorNode` |
| ブロック種別 | ブロックの種類を表す列挙型 | `BlockKind` |
| インラインラン | ブロック内のインライン書式区間（bold, italic 等） | `InlineRun` |
| アクティブブロック | カーソルが位置するブロック。Markdown ソースが表示される | `activeBlockID` |
| フォーカスモード | アクティブブロック以外を淡色表示にする集中執筆モード | `isFocusModeEnabled` |
| テーマ | エディタの配色定義 | `EditorTheme` |
| デリミタ | Markdown の書式記号（`**`, `*`, `` ` ``, `#` 等） | `delimiterColor` |
| スナップショット | ドキュメント保存用のスレッドセーフなテキストコピー | `_snapshotText` |
| アセットディレクトリ | 画像等を保存するドキュメント隣接ディレクトリ | `assetsDirectoryURL()` |

## コンポーネントごとのビジネス目的

| コンポーネント | ビジネス目的 |
|---------------|-------------|
| `ShoechooApp` | アプリケーションエントリーポイント、ドキュメントライフサイクル管理 |
| `MarkdownDocument` | ファイル読み書き、スナップショット管理、アセットディレクトリ管理 |
| `EditorViewModel` | エディタ状態管理、フォーマットコマンド発行、統計情報 |
| `EditorNodeModel` | パース結果のブロックツリー管理、アクティブブロック解決 |
| `MarkdownParser` | swift-markdown AST → EditorNode ツリー変換 |
| `SyntaxHighlighter` | WYSIWYG表示の実現（属性適用） |
| `ShoechooTextView` | フォーカスモード描画、タイプライタースクロール、画像D&D、自動ペア |
| `WYSIWYGTextView` | SwiftUI↔AppKit ブリッジ、ハイライトスケジューリング |
| `ExportService` | HTML/PDF エクスポート |
| `ImageService` | 画像インポート（セキュリティ検証込み） |
